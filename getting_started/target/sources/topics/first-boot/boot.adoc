
=== Booting the Server


:include_filename: getting_started/topics/first-boot/boot.adoc
include::../templates/header.adoc[]


To boot the {project_name} server, go to the _bin/_ directory of the server distribution.

.Standalone Boot Scripts
image:{project_images}/standalone-boot-files.png[]

To boot the server:

.Linux/Unix
[source]
----
$ .../bin/standalone.sh
----

.Windows
[source]
----
> ...\bin\standalone.bat
----
