Jetty support

  • Install Jetty WTP plugin.
  • Download Jetty 8.
  • Add Jetty server to Eclipse.
  • Open the Servers view
  • Open your server's config (right-click -> Open, or F3)
  • On the Overview tab, click Open launch configuration"
  • Classpath tab -> add external jars
  • Add jars from <jetty>/lib/annotations directory.
  • Add to web.xml: 
    <listener>
	<listener-class>com.sun.faces.config.ConfigureListener</listener-class>
</listener>
  • Jetty Maven Plugin
