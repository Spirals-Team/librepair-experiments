Custom Plugins for Apache JMeter™

This project is an independent set of plugins for Apache JMeter, the popular Open-Source load and performance testing tool.

Latest Update

Version 1.3.1 was released on the 12th of October, 2015. It is mostly a maintenance release that fixes issue with Ultimate Thread Group. See Changelog page for the list of all other changes.

The prerequisite is to run JMeter 2.12 or above with a Java 7. Follow installation procedure after download.

Usage

Once installed, you can use the plugins as standard JMeter objects, they will appear in the JMeter menus. You can save, copy to clipboard or export the graphs data by right-clicking them, and modify the rendering in different ways. All possibilities are described here.
