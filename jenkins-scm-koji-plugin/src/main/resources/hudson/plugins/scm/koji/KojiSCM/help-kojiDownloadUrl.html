URL for RPM download. For example:
http://koji.fedoraproject.org/packages/:
You can put more urls separated by space. first found first served.
http://koji.fedoraproject.org/packages/ http://custom.koji:DPORT/
Warning: if build checkouted from later topUrl is found in earlier downloadUrl. It is downloaded instead of later! (See fixme in KojiBuildDownloaderdownloadRPM).
